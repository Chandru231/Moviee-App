# CineSearch - Movie Discovery Platform

A modern, responsive movie discovery web application built with React and powered by The Movie Database (TMDB) API.

# Setup Instructions

1. Clone the repository

2. Install dependencies

3. Configure API Key - Create a .env file

   VITE TMDB API KEY= My tmdb api key (paste here)

4. Restart the project.

   npm run dev
